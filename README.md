# 📦 Инвентаризация — Мониторинг в реальном времени

Веб-приложение для отслеживания прогресса инвентаризации товарных групп в магазине. Показывает в реальном времени, какие группы уже посчитаны, а какие ещё ждут своей очереди.

## Возможности

- 📊 **Дашборд** — общая статистика: сколько групп посчитано, процент готовности, расхождения
- 🔄 **Real-time обновления** — WebSocket подключением для мгновенного отображения изменений
- 📁 **Загрузка файлов** — drag & drop или выбор CSV/Excel файла
- 🔍 **Фильтры и поиск** — по магазину, категории, статусу, названию группы
- 🔔 **Уведомления** — всплывающие нотификации при обновлении данных
- ⚠️ **Расхождения** — визуальное выделение излишков, недостач и брака
- 📱 **Адаптивный дизайн** — работает на десктопе и мобильных устройствах

## Технологии

- **Бэкенд:** Python + FastAPI + WebSocket
- **Фронтенд:** Vue 3 + Vite + Pinia + TailwindCSS
- **Парсинг:** pandas (CSV, Excel)

## Быстрый старт

### Требования

- Python 3.10+
- Node.js 18+
- npm или yarn

### 1. Запуск бэкенда

```bash
# Переходим в папку бэкенда
cd backend

# Создаём виртуальное окружение
python -m venv venv

# Активируем (Windows)
venv\Scripts\activate

# Активируем (Linux/Mac)
# source venv/bin/activate

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем сервер
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Бэкенд будет доступен на `http://localhost:8000`

**API документация:** `http://localhost:8000/docs`

### 2. Запуск фронтенда

```bash
# Переходим в папку фронтенда
cd frontend

# Устанавливаем зависимости
npm install

# Запускаем dev-сервер
npm run dev
```

Фронтенд будет доступен на `http://localhost:3000`

### 3. Загрузка данных

Есть два способа:

1. **Через интерфейс** — перетащите CSV/Excel файл на страницу или нажмите кнопку загрузки
2. **Авто-мониторинг** — положите файл в папку `backend/uploads/`, бэкенд автоматически его обработает

## API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/health` | Проверка здоровья |
| POST | `/api/upload` | Загрузка CSV/Excel файла |
| GET | `/api/groups` | Список групп (с фильтрами) |
| GET | `/api/stats` | Общая статистика |
| GET | `/api/stores` | Список магазинов |
| GET | `/api/categories` | Список категорий |
| WS | `/ws` | WebSocket для real-time обновлений |

### Параметры фильтрации `/api/groups`

- `store` — фильтр по магазину (например, `S187`)
- `category` — фильтр по категории
- `status` — статус: `counted`, `partial`, `not_counted`
- `search` — поиск по названию группы
